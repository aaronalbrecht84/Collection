# Module 8 Journal
The goal of project 3 was to create a program that could read a file, count how many times each string appeared in that file, display a menu that allows the user to either get the frequency of a desired string, display the frequency for all strings, display the frequency of all strings as a histogram, or end the program.  By ending the program, a frequency.dat file is created to listing all the strings and their corresponding frequencies.  I feel that one of the I did particularly well was creating methods to handle each part to minimize the amount of coding in the main segment.  While this program functions well, this could be enhanced to create greater adaptability.  Overall there wasn't any part that was particularly difficult.  I did initially have an issue importing the file and creating the inital map, but that was resolved when I took a quick break to mull over the issue and realized the count of each string wasn't totaling correctly.  The skills I learned with this program that can easily be utilized elsewhere are reading data from an input file and creating an output file.  I tried to make this program maintainable, readable, and adaptable by creating methods to handle different parts so there are no extensive blocks of code and keeping everything organized with comments indicating what the code's purpose was.
